Requirements

Before installing LIDE, ensure your system meets these requirements:

  • Python 3.7 or higher
  • Flask 2.0.1
  • Modern web browser (Chrome, Firefox, Safari, Edge)

Quick Start

Get LIDE up and running in minutes with these simple steps:

1. Clone the repository:

git clone https://github.com/gtsrhythm/Lide.git
cd lide

2. Run the setup script:

chmod +x run.sh
./run.sh

3. Open your browser and navigate to:

http://localhost:5000

Development Setup

For development purposes, follow these additional steps:

1. Create a virtual environment:

python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

2. Install dependencies:

pip install -r requirements.txt

3. Run in development mode:

./run.sh dev

Deployment

For production deployment, you have two options:

1. Standard Deployment:

./run.sh prod

2. Using Docker:

docker build -t lide .
docker run -p 8000:8000 lide
