Requirements

Before installing LIDE, ensure your system meets these requirements:

  • Python 3.7 or higher
  • Flask 2.0.1
  • Modern web browser (Chrome, Firefox, Safari, Edge)

Quick Start

Get LIDE up and running in minutes with these simple steps:

1. Clone the repository:

git clone https://github.com/gtsrhythm/Lide.git
cd lide

2. Run the setup script:

chmod +x run.sh
./run.sh

3. Open your browser and navigate to:

http://localhost:5000

Development Setup

For development purposes, follow these additional steps:

1. Create a virtual environment:

python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

2. Install dependencies:

pip install -r requirements.txt

3. Run in development mode:

./run.sh dev

Deployment

For production deployment, you have two options:

1. Standard Deployment:

./run.sh prod

2. Using Docker:

docker build -t lide .
docker run -p 8000:8000 lide

Scripting Guide

Learn how to extend and automate LIDE using its powerful scripting capabilities.

Automation Features

LIDE supports custom automation scripts within .mne files using a simplified scripting syntax. Scripts can be created using either code fences or triple colons:

// Using code fences:
```
// Your automation code here
```

// Using triple colons:
:::automation
// Your automation code here
:::

Scripts can be used to:

  • Automate repetitive tasks
  • Add custom commands
  • Extend editor functionality
  • Create custom workflows

Basic Syntax

Simple Commands

create file.txt
create src/
delete file.txt
print "Hello, World!"

Functions

fn setup() {
    print "Starting setup..."
    createFile("config.json")
}

fn cleanup() {
    print "Cleaning up..."
}

Tasks

task "scaffold" {
    name: "myproject"
    template: "basic"
}

Built-in Functions

File Operations

createFile(path, content?) - Create a new file
deleteFile(path) - Delete a file
readFile(path) - Read file content
writeFile(path, content) - Write content to file

Directory Operations

createDirectory(path) - Create a new directory
deleteDirectory(path) - Delete a directory and its contents

Output Functions

print(...messages) - Print normal messages
error(...messages) - Print error messages in red
success(...messages) - Print success messages in green
clear() - Clear the output terminal

Utility Functions

sleep(ms) - Pause execution for specified milliseconds

Running Automations

Follow these steps to create and run automations in LIDE:

  1. Create a file with the .mne extension
  2. Add your automation code using the syntax above
  3. Run the automation using:
    • Click the "Run" button in the automation block
    • Use the keyboard shortcut Ctrl+R
    • Click the play button in the file tab

Best Practices

  • Use descriptive function names
  • Add error handling with try/catch blocks
  • Provide user feedback with print functions
  • Break down complex tasks into smaller functions
  • Use sleep() to control execution timing
